※ 지역별 축제 소개

월 기간 : 8월 ~ 12월(5개월 간)

지역 : 서울, 부산, 대구, 인천, 광주, 대전, 울산, 경기, 강원(9개 지역)



프론트 엔드 기능

1. 축제 광고 삽입은 카테고리 별로 최소 3개 최대 5개 사용(jQuery )

2. 지역은 한국 전체 / 분야는 문화, 온라인 축제, 관광 축제, 공연 등으로 구성 
   정렬은 평점, 인기, 최신, 거리 등으로 구성(select, 콤보 박스)(필터 역할)

3. 화살표는 월을 넘기는 화살표로 페이지를 넘기는 것을 의미, 달력엔 축제 일정을 표기
   밑에는 언더라인으로 기한 표시(혼란 여부로 선택사항), 클릭시 상세 설명과 이미지
   맨 밑에는 평점과 리뷰를 달 수 있는 시스템(html, css, flex-direction: row-reverse)(동적)

   달력안에 있는 축제명을 클릭하면 축제를 자세히 설명해주고 예매팁 혹은 축제에 관한 
   유튜브, 블로그 등을 보여줌

4. 이메일 주소와 sns 주소를 연결시켜 소통 창구를 조성하고 Q&A 칸 생성 


백엔드 기능

1. 평점과 리뷰 달은 데이터를 연동, 회원에 한하여 관심 분야, 관심 지역을 설정할 수 있도록 함
   설정 지역에 관심 분야 축제가 올라오면 수신 동의 후 알림이 가게 함

2. Q&A는 비밀글(시간되면)과 공개글 설정이 가능하고 관리자에게 이미지, 텍스트, 음성 관계 없이 전송
   가능하도록 설정

   (축제를 다녀온 후 이 웹페이지에 후기를 남기면 포인트를 주고 
   포인트를 통해 예매 할인권이라던지 혜택을 줄 수 있도록 함)
